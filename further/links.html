# Quick link

收集 web 领域常用的网站。

# 文献或指南

  • MDN在新窗口打开: Mozilla 维护的开发文档, 可能是最齐全的 web 文档
  • caniuse在新窗口打开: 查看指定 CSS 在各大浏览器中的兼容程度
  • statcounter在新窗口打开: 查看世界浏览器市场份额

# Animation

动画相关的网站。

  • css 动画库在新窗口打开
  • stylie在新窗口打开: css3贝塞尔曲线可视化工具

# 性能相关

测试或研究网页性能相关的网站。

  • webpagetest: 网页性能测试
  • jsperf在新窗口打开: 可以在不同的浏览器测试 JavaScript 的性能
  • csstriggers在新窗口打开: 一个CSS属性影响渲染的参考表(回流与重绘)

# Online tools

一些常用的在线工具。

  • 草料二维码在新窗口打开
  • tinypng在新窗口打开: 压缩图片
  • 在线工具在新窗口打开: 在线加解密, 转base64等
  • jsbin在新窗口打开: 在线编辑器, 可以将自己的代码保存发给别人
  • 可视化正则表达式在新窗口打开 将正则表达式转为可视化图表, 让你一眼能看懂这段正则是干啥的
  • 代码可视化网站在新窗口打开 可将代码按步骤执行，分段可视化展示，并允许将可视化代码嵌入网站上
